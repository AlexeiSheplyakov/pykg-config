pykg-config by gbiggs

Python drop-in replacement for pkg-config

Dependencies

Python 2.6

Install

On Windows, run the available installer. On other systems, run "python setup.py install" from the source directory (this also works on Windows).

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/gbiggs/pykg-config
